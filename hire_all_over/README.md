<H2>NPCBOT file to add hire npcs in starting areas and cities:</H2>  

Add this to the WORLD database.  
This will add interactable npcs in starting zones and cities where you can purchase spawned npcbots (not wanderingbots).  

<B>Add npcs:</B>  
- HireAllOver.sql  
  
<B>Remove npcs:</B>  
- HireAllOverRemove.sql will erase everything added in creature table from the file HireAllOver.sql
